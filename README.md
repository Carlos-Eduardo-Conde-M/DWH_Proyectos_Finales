# 📚 Proyectos Finales - Data Warehouse (DWH)

Bienvenidos al repositorio de **Proyectos Finales de Data Warehouse (DWH)**.

## Objetivo

Este repositorio tiene como finalidad centralizar y almacenar todos los proyectos finales desarrollados por los estudiantes de la materia de Data Warehouse.

Cada estudiante deberá subir todos los archivos, documentos, scripts, bases de datos, dashboards, reportes y demás recursos utilizados durante el desarrollo de su proyecto.

---

## Estructura del Repositorio

Cada estudiante debe crear una carpeta con su **nombre completo en MAYÚSCULAS**, siguiendo el siguiente formato:

```text
/
├── JUANITO ALCACHOFA/
├── BRYAN IBAN/
├── CARLOS CONDE/
└── ...
```

### Contenido de la carpeta

Dentro de su carpeta personal deben incluir:

* Modelo dimensional.
* Scripts SQL.
* Archivos CSV o datasets utilizados.
* Documentación del proyecto.
* Reporte final.
* Dashboard (Power BI, Tableau, etc.).
* Presentación.
* Cualquier otro recurso relacionado con el proyecto.

---

## Recomendaciones

* Mantener una estructura organizada de carpetas y archivos.
* Verificar que todos los recursos necesarios estén incluidos.
* Utilizar nombres descriptivos para los archivos.
* Asegurarse de que los dashboards y scripts funcionen correctamente antes de enviarlos.

---

## Fecha de Entrega

La fecha límite de entrega será comunicada por el docente.

---

## Soporte

Ante cualquier inconveniente con Git, GitHub o el proceso de envío mediante Pull Request, comunicarse con el docente o ayudante de la materia.

¡Éxitos en la presentación de sus proyectos finales! 🚀
